# ModHossHate
Setup a `.env` file with the following content

```
TWITCH_BOT_USERNAME=
TWITCH_OAUTH_TOKEN=

FIREBASE_API_KEY=
FIREBASE_AUTH_DOMAIN=
FIREBASE_DATABASE_URL=
FIREBASE_PROJECT_ID=
FIREBASE_STORAGE_BUCKET=
FIREBASE_MESSAGING_SENDER_ID=
FIREBASE_APP_ID=
FIREBASE_MEASUREMENT_ID=
```

Run the bot
```
npm start
```
